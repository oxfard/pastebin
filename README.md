Yii 1.1 Pastebin Template
============================
Шаблон простого приложения Yii 1.1. Реализует базовый функционал сервиса Pastebin.com.

Требования
----------
Минимальными требованиями для работы приложения является PHP 5.6, Yii framework 1.1+ (рекомендуется ставить последнюю версию 1.1.22), MySQL 5.7+ (протестировано на 5.7.28) и веб-сервер (протестировано на Apache 2.2.15). Для корректного отображения URL потребуется mod_rewrite.

Установка
---------
После скачивани содержимое каталога необходимо распокавать в папку предварительно настроенного виртуального хоста веб-сервера. Дамп бд находится в protected/data/mysql.schema.sql
На сервере должен быть установлен Yii framework.

Настойка Приложения
-------------------
В строке $yii=dirname(__FILE__).'/../framework/yii.php' файла index.php надо указать текущее расположение yii.php. Он располагается на верхнем уровне каталога фреймворка.  
Все настройки производятся в protected/config/main.php. Там необходимо прописать настройки подключения к бд.
~~~
'components'=>array(
		...
		'db'=>array(
			'connectionString' => 'mysql:host=localhost;dbname=pastebin',
			...
			'username' => 'pastebin',
			'password' => '111qwert111',
			...
		),
~~~
В MySQL необходимо создать соответствующую базу и пользователя с правами на чтение, запись и модификацию.
